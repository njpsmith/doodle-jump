# Modules / Files

1. index.html – Canvas or DOM container
2. game.js – Main game loop & state manager
3. player.js – Player entity, physics, input handling
4. platform.js – Platform entity, procedural generation logic
5. enemy.js – Optional moving hazards
6. powerup.js – Items player can collect
7. input.js – Keyboard, touch, and accelerometer input abstraction
8. ui.js – Score display, menus, game over screens
9. storage.js – High score persistence (localStorage/IndexedDB)
10. utils.js – Math helpers, randomization, collision detection
11. renderer - Canvas rendering
